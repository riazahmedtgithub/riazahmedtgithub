Datascience projects

1. House Price Prediction using XGBoost and ANN.

  • Real estate value prediction model.
  • Data source: Kaggle - Housing Prices.
  • Includes feature engineering, visualization, XGBoost & ANN modeling.
  • GridSearchCV & RandomizedSearchCV for hyperparameter tuning.
  • Final model recommendation based on metrics comparison.

2. Predicting Neurodegenerative Disease using KNN and PCA.

  • Prediction based on test results like Albumin, Sodium, Urine PH, etc.
  • Data source: Bellevue University.
  • Used clustering, silhouette analysis, PCA for visualization.
  • Eigenvalues and eigenvectors calculated post PCA.

3. Sentiment Analyzer using NLTK.

  • Movie review sentiment prediction using TextBlob and Vader.
  • Includes tokenization, stopword removal, stemming.
  • TFIDF Vectorization and models: Logistic Regression, SVM.
  • Final model chosen based on evaluation metrics.

4. Breast cancer prediction using linear model in R language.

  • Predicting the chances of breast cancer depending upon the age, tumour size, characteristic, menopause stage etc.
  • Data Source: Oncology institute.
  • Used R language, ggplot, tidyverse, broom, caret and metrics package.
  • Converting the variables to factor.
  • Used glm - linear model and stepAIC function to determine the optimum predictor variables.

5. Predicting customer churn for subscription-based program.

  • Predicting whether the customer will churn or not, by using the existing data of the subscription provider.
  • Data Source: Kaggle, https://www.kaggle.com/datasets/safrin03/predictive-analytics-for-customer-churn-dataset.
  • Used Chi Squared test of independence to compare how the churn is associated with other variables.
  • Used Standard scaler, One-Hot encoding and SMOTE method has been used.
  • Dimensionality reduction using PCA.
  • Analysis of linear separation of data based on Principal components and comparison of PCA vs RBF.
  • Model Building using RandomizedSearchCV.

6. Implementing Retrieval Augmented Generation (RAG) for LLM model.

  • RAG implementation using ChromaDB for gpt-4o-mini openai model.
  • Used langchain and Wikipedia libraries.
  • RecursiveCharacterTextSplitter has been used for splitting.
  • OpenAIEmbeddings class has been used to generate vector embeddings.

7. Personalized gift recommendation GenAI model and streamlit.

  • Fine tuning a personal GenAI model using gpt-4o-mini to offer personalized gift suggestions.
  • OpenAI model has been used and fine tuning job has been created using an uploaded training file.
  • Fine tuning job information retrieval and metrics evaluation.
